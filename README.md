# Casa Chidísima 3D

Proyecto estático listo para GitHub Pages.

## Controles

- Clic en el visor: activar cámara con mouse.
- ESC: liberar mouse.
- W / Flecha arriba: avanzar.
- S / Flecha abajo: retroceder.
- A / Flecha izquierda: izquierda.
- D / Flecha derecha: derecha.
- Q: subir.
- E: bajar.
- Shift: moverte más rápido.

## Publicación

Sube todo a la raíz del repositorio y activa:

Settings > Pages > Deploy from a branch > main > / root
